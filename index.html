Oque é uma variavel?
Ela funciona como um espaço na memória do computador onde você pode armazenar dados que podem ser alterados durante a execução do programa.

Oque é um loop?
È uma estrutura que permite a repetição de um bloco de código várias vezes, até que uma condição específica seja atendida.
